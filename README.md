# Ajay Purbiya | MERN Stack Developer Portfolio

[![GitHub Repo stars](https://img.shields.io/github/stars/ajaypurbiya/portfolio?style=social)](https://github.com/ajaypurbiya/portfolio)

Full-stack portfolio built with **MongoDB**, **Express**, **React**, **Node.js** (MERN).

## 🚀 Features
- ✅ Public portfolio: Hero, Skills, Dynamic Projects
- ✅ JWT Authentication & Protected Dashboard
- ✅ Image Upload for projects
- ✅ Responsive Tailwind CSS + Framer Motion animations
- ✅ MongoDB backend with CRUD operations

## 🛠️ Quick Start

1. **Clone repo**
   ```bash
   git clone https://github.com/ajaypurbiya/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm run install:all
   ```

3. **Backend (.env required)**
   ```
   cd backend
   npm run dev  # http://localhost:5000
   ```

4. **Frontend**
   ```bash
   cd frontend
   npm start    # http://localhost:3000
   ```

## 📱 Deployed
- Frontend: [Vercel/Netlify](https://portfolio-ajay.vercel.app)
- Backend: [Render/Railway](https://portfolio-backend.onrender.com)
